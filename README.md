# dockerize-ruby
A ruby gem for dockerize your app

## Build steps

1. gem build

## Install

```
gem install dockerize-ruby
```

## Running

1. export your chatGPT api key:
```
export OPENAI_API_KEY=sp-yourapikey
```

2. generate a Dockerfile:
```
dockerize-ruby dockerfile
```

## Testing

```
rake
```
